## Dashboard

The main dashboard shows:
- Total Value Locked across all vaults
- Active Constraints count
- Uptime / Health status
- Individual vault cards with TVL, APY, risk level, and status

## Vault Detail Page

Each vault has a dedicated page with:

- **Metric Cards**: NAV, Policy Version, Pending Actions, Queued Cycles
- **Constraint State**: Current values for drawdown, leverage, position caps
- **Agent Status**: Active cycle, last completed, permission level
- **Pending Actions**: Actions awaiting your approval
- **Recent Activity**: Indexed events from the registry
- **Agent Logs & Reasoning**: On-chain decisions and off-chain reasoning from the orchestrator

## Explorer

Each vault page links to Solana Explorer for direct on-chain inspection.
