# Krypton Marketing & Landing

## Positioning

Krypton is not an AI hedge fund. Frame as: policy interpreter + constrained execution layer.

Forbidden: alpha, returns, managed, fund, guaranteed.

## Ika copy correction

Do NOT claim encrypted balances or ZK proof-of-reserves in V1.

DO claim: Ika dWallet MPC signing — agents never hold keys; Constraint Engine gates co-signing.

## Landing structure

1. Hero + pipeline strip (policy → agents → constraints → vault)
2. Differentiation table (One-Pager)
3. Architecture primitives
4. Integration surface
5. CTA: Request private beta access

## Visual system

Colors: #0E1116 bg, #FFB02E policy accent, #5EE6D9 signing accent.

Fonts: Space Grotesk, IBM Plex Sans, IBM Plex Mono.
